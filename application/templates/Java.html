Java is a general purpose, high-level programming language developed by Sun Microsystems. A small team of engineers, known as the Green Team, initiated the language in 1991. Java was originally called OAK, and was designed for handheld devices and set-top boxes. Oak was unsuccessful, so in 1995 Sun changed the name to Java and modified the language to take advantage of the burgeoning World Wide Web.

Getting started with Java:

Object − Objects have states and behaviors. Example: A dog has states - color, name, breed as well as behavior such as wagging their tail, barking, eating. An object is an instance of a class.
Class − A class can be defined as a template/blueprint that describes the behavior/state that the object of its type supports.
Methods − A method is basically a behavior. A class can contain many methods. It is in methods where the logic's are written, data is manipulated and all the actions are executed.

If statements:

The Structure of IF statements in Java are shown in the example:
An if statement can be followed by an optional else statement, which executes when the Boolean expressions is false. 

         int age = 35;
         int checkage = 30;
         if(age > checkage){
             System.out.println("You can't drive");
         }

The if...else if...else statement:

When using if, else if, else statements there are a few points to keep in mind.
An if can have zero or one else's and it must come after any else if's. An if can have zero to many else if's and they must come before the else. Once an else if succeeds, none of the remaining else if's or else's will be tested. 

 int age = 21;
	  int checkage = 30;
	  if(age > checkage){
		  System.out.println("You can't drive");
	  }
	  else if(age == checkage){
		  System.out.println("You can't vote");
	  }
	  else {
            System.out.println("your age is:" + age+ " so you can't do either");
	  }

For loop

A loop statement allows us to execute a statement or group of statements multiple times.A for loop is useful when you know how many times a task is to be repeated, as it is a repetition control structure that allows you to efficiently write a loop that needs to be executed a specific number of times. 

for(int number=2;number<=8;number++){
		System.out.println(number);
		}

While loops

The key point of the while loop is that the loop might not ever run. When the expression is tested and the result is false, the loop body will be skipped and the first statement after the while loop will be executed.

int number = 2;
		while (number<=8){
			System.out.println (number);
			number++;
		}

}

Do...While loops

A do...while loop is similar to a while loop, except that a do...while loop is guaranteed to execute at least one time. The Boolean statement in a do...while loop appears at the end of the loop, so the loop will always run once before the Boolean is tested.

int number = 2;
		do {
			System.out.println(number);
		number++;
		}
		while (number<=8);
	}
