SQL Update Statement

The update statement modifies certain pieces of data in a table. 

UPDATE table_name
SET column1=value1,column2=value2,...
WHERE some_column=some_value;
FROM table_name;


The WHERE clause is important because it specifies what records it is talking about. If the WHERE clause is not included in the statement, then it assumes that all records will be updated. 


UPDATE Customers
SET City='Hamburg'
WHERE CustomerID=1;
