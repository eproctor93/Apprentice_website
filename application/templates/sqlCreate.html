Creating a Database

You can't do any SQL statements until you have actually created a database first. To create a database, you use the following syntax.

CREATE DATABASE dbname


Creating Tables

Without the CREATE TABLE statement, you won't be able to create tables in a database. Each table must have a name, row and a column. You start off by naming your table and then specifying the names of the columns. You also must say what datatype the column can hold.


CREATE TABLE table_name
(
column_name1 data_type(size),
column_name2 data_type(size),
column_name3 data_type(size),
...
);

Data Types

  • VARCHAR
  • CHAR
  • Boolean
  • Integer
  • Decimal
  • Float
  • Date
  • Time
