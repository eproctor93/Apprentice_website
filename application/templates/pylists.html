Lists

If you come across a data structure, then arrays is one of the most fundamental ones that you come across in Python. It is not native to Python, but it has more of a general list. Before you come across arrays, you will definitely come across lists. They are like an ordered collection of items and you can have a list of any data type.

How to create a list

To create a list, you start by creating a list variable naming it whatever you want. You define it by adding different values in square brackets, separated by commas. List indices always start with an index of 0. So the first item of the list will have an index of 0 and not 1. Next item would have an index of 1 and so on. 


    prog1 = ["Python", "HTML", "CSS", "Javascript", "Ruby"];
    prog2 = [1, 2, 3, 4, 5];
    prog3 = 1995, "Plymouth", 1997, "England", 1999];

Accessing things from the list

If you wanted to access things fron the list, you would say the list name, followed by the index number you wanted to look at. 


  print "prog1[0]:" , prog1[0]  # Prints what is at index 0 in the list prog1. In this case it would print Python, since it is at index 0.
  print "prog2[1:5]:", list2[1:5] # This prints index 1 all the way to index 4 in prog2. So that would be 2,3,4,5.

Updating and deleting lists

You can modify things from the list by using the following items.

Item What does it do?
append(): Adds a new entry to the end of the list
clear(): Removes all entries from the list
copy(): Creates a copy of the current list and places it in a new list.
extend(): Adds items from existing list and into the current list.
insert(): Adds a new entry to the position that it is specified in the list.
remove(): Removes an entry from the list
