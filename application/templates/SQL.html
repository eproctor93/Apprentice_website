What is SQL used for?

It is used across web frameworks and all database applications. SQL gives you the freedom to explore your data a lot more. There are concepts that apply to nearly every data storage system and learning SQL should help you understand more about data storage systems.

The other things that SQL can do are:

  • Execute queries against a database
  • Find data from a database
  • Can insert, update and delete records from a database.
  • Can create new databases
  • Can create new tables
  • And lots more

Key Concepts

A database contains one or more tables which are identifed by a name (eg customers). Most actions need to be done on a database with SQL statements. One example is that this SQL statement selects all the records in the "Customers" table.

SELECT * FROM Customers;

Every SQL statement ends in a semicolon, because it allows more than one SQL statement to be executed. SQL has one major difference from all programming languages and that is SQL keywords are NOT case sensitive. For example, select is the same as SELECT.

We will cover in more detail in separate pages about the commands. But, here are some of the most important SQL commands

Command Description
SELECT Extracts data from a database
UPDATE Updates data in a database
DELETE Deletes data from a database
INSERT INTO Inserts new data into a database
CREATE TABLE Creates a new table
ALTER TABLE Modifies a table
DROP TABLE Deletes a table

SQL SELECT Statement

This statement is used to select data from a database. 

SELECT column_name,column_name
    FROM table_name;
SELECT CustomerName, City FROM Customers;

SQL Update Statement

The update statement modifies certain pieces of data in a table. 

UPDATE table_name
SET column1=value1,column2=value2,...
WHERE some_column=some_value;
FROM table_name;


The WHERE clause is important because it specifies what records it is talking about. If the WHERE clause is not included in the statement, then it assumes that all records will be updated. 


UPDATE Customers
SET City='Hamburg'
WHERE CustomerID=1;


SQL Delete Statement

The SQL delete Statement is where you delete certain things from a table. It is very important that you use the delete statement with caution, because you don’t want to delete important data or a whole table. You can't undo this statement, so it is important that you know what you are doing with this statement. 

DELETE FROM Customers
WHERE CustomerName= 'Alfreds Futterkiste' AND ContactName='Maria Anders';

SQL INSERT INTO Statement

This statement is used to insert new records in a table. There are two ways in which you can write these statements. The first way is that you don’t have to specify the column where the data is inserted. It assumes it will be updated to all columns and the values have to be in the right order. Another way of doing it is that you specify both the column name and the values to be inserted. 

INSERT INTO table_name
VALUES (value1,value2,value3,...);
INSERT INTO Customers (CustomerName, ContactName, Address, City, PostalCode, Country)
VALUES ('Cardinal','Tom B. Erichsen','Skagen 21','Stavanger','4006','Norway');

Creating a Database

You can't do any SQL statements until you have actually created a database first. To create a database, you use the following syntax.

CREATE DATABASE dbname


Creating Tables

Without the CREATE TABLE statement, you won't be able to create tables in a database. Each table must have a name, row and a column. You start off by naming your table and then specifying the names of the columns. You also must say what datatype the column can hold.


CREATE TABLE table_name
(
column_name1 data_type(size),
column_name2 data_type(size),
column_name3 data_type(size),
...
);
